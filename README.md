# Charts Flutter

简单美观流畅的动画图表。基于https://github.com/merixstudio/mrx-flutter-charts该项目做的修改。

## Supported charts

### Bar

原本功能
![Screenshot1](https://raw.githubusercontent.com/fucsusu/lwl-flutter-charts/develop/screenshots/chart_bar.gif)
添加目标功能：  
1.坐标轴刻度对齐  
2.支持背景颜色的柱状图  
3.自定义单个柱子的颜色  
4.瀑布图的支持
